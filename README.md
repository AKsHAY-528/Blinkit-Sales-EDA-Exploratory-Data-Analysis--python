# Blinkit Sales Data Analysis using Python and Pandas

## Project Overview

This project focuses on analyzing Blinkit retail sales data using Python and the Pandas library. The goal of the analysis is to explore sales trends, understand product performance, and extract meaningful insights from the dataset using exploratory data analysis techniques.

The project demonstrates how raw retail data can be cleaned, processed, and analyzed to support data driven decision making.

---

## Objectives

* Perform exploratory data analysis on Blinkit sales data
* Understand product level and outlet level performance
* Identify patterns in sales distribution
* Clean and preprocess raw dataset using Pandas
* Generate insights that can help improve retail performance

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn (if used)
* Jupyter Notebook

---

## Dataset Description

The dataset contains information related to Blinkit retail sales including:

* Item identifiers
* Item type and category
* Outlet establishment details
* Outlet size and location
* Item visibility
* Item weight
* Sales values

These attributes allow analysis of how different factors influence product sales.

---

## Key Analysis Performed

The notebook includes the following analytical steps:

### Data Cleaning

* Handling missing values
* Checking data types
* Removing inconsistencies

### Exploratory Data Analysis

* Distribution of product categories
* Sales comparison across outlet types
* Analysis of outlet size and location impact on sales
* Item level performance insights

### Data Visualization

Visualizations are used to better understand trends and patterns in the dataset.

## How to Run the Project

1. Clone this repository
2. Install required libraries

```
pip install pandas numpy matplotlib seaborn
```

3. Open the Jupyter Notebook

```
jupyter notebook
```

4. Run the notebook cells to reproduce the analysis.
